Detailed Project Abstract

Prodo was engineered to solve "Academic Fragmentation"—the stress caused by managing educational responsibilities across disconnected platforms. By centralizing schedules, tasks, and weighted grade tracking, Prodo allows students to focus on performance.

The application leverages a real-time data synchronization engine. When a student completes a task on the Kanban Board, the system triggers a server-side recalculation of their completion progress.

Optimized Capabilities

📋 Kanban Board

Visual workflow management tracking assignment lifecycles from pending to archived.

📊 GPA Engine

Automated grade point calculation based on credit hours and custom scales.

⏱️ Focus Timer

Integrated Pomodoro logic designed to enhance student information retention.

🔒 Note Vault

Secure, authenticated storage for academic documentation and class archives.
